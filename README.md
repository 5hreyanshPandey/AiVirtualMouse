
# AI Virtual Mouse & Media Control System

An AI-powered gesture recognition system that allows users to control the mouse cursor and manage media playback using hand movements. This project eliminates the need for physical input devices and provides a touch-free, intuitive interaction experience.

##  Features

* Mouse movement using hand tracking
* Left & Right click using gestures
* Scroll control via finger motion
* Media control using gestures:
  * Play / Pause
  * Volume Up / Down
  * Next / Previous
* Works with:
  * YouTube
  * VLC Media Player
  * Spotify
  * All browser-based media platforms
* Real-time performance with high accuracy

## Tech Stack

* Python
* OpenCV
* MediaPipe
* PyAutoGUI
* Gesture Recognition (AI)
  
##  Objective

To create a smart Human-Computer Interaction system that replaces traditional mouse and keyboard controls with AI-based hand gestures for both navigation and media control.

##  System Requirements

* Python 3.7+
* Webcam
* Windows OS
* Minimum 4GB RAM

##  Output

Users can fully control the system cursor and multimedia platforms using only hand gestures captured through a webcam.

##  Future Enhancements
